Project 8: Ember Tutorial

This project goal is to learn how to build a basic application using Ember. While working on the project, I learned how to use some key CLI commands relating to Ember. Also I was able to use UI component and define route to build on my application.

The project consists of two pages and a component of a list of people scientists' name. When a user click on it, an event handler will triggers to display the clicked name in a pop-up.
